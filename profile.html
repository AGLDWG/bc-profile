Definition

This is a profile that mandates minimal metadata for the cataloguing of (sub)catalogues.

It is a profile of the well-known DCAT, the semantic data cataloguing vocabulary standard and also & schema.org, the widely-used, general-purpose web object description vocabulary which also has data cataloguing elements.

This profile is used by multiple Australian government and international catalogues that contain other catalogues and is entirely generic, i.e. the additional constraints are not Australian or government-specific but just ensure certain technical model capabilities in the data.
